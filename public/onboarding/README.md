## 이미지 준비 가이드

### 스크린샷 촬영 시 고려사항
1. **해상도**: 고해상도로 촬영 후 WebP로 최적화
2. **크기**: 600x400px 또는 16:9 비율 유지
3. **품질**: WebP 포맷으로 압축하여 파일 크기 최적화
4. **내용**: 각 슬라이드의 핵심 기능이 명확히 보이도록 촬영

### 이미지 최적화 도구
- **온라인**: TinyPNG, Squoosh.app
- **로컬**: ImageOptim, FileOptimizer
- **포맷 변환**: cwebp (Google WebP 도구)
